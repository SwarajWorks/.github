# ✨ About

FlowAgent is an AI-powered workflow automation platform that helps businesses and developers automate repetitive tasks using intelligent AI agents and seamless integrations.

Whether you're building internal tools, automating business processes, or connecting third-party services, FlowAgent makes automation simple.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/SwarajWorks/flowagent.git
```

Move into the project

```bash
cd flowagent
```

Install dependencies

```bash
npm install
```

Start development

```bash
npm run dev
```
